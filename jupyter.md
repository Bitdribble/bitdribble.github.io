#### Docs
* [jupyter.org](https://jupyter.org)
  * [Install](https://jupyter.org/install)
    * `pip install jupyterlab notebook`
    * Put `~/.local/bin` in the `PATH`
    * Execute `jupyter lab` to start the lab, `jupyter notebook` to start the notebook
