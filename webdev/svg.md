[SVG](https://en.wikipedia.org/wiki/Scalable_Vector_Graphics) (Scalable Vector Graphics) is an XML-based vector image format for two-dimensional graphics.

* MDN: [SVG Tutorial](https://developer.mozilla.org/en-US/docs/Web/SVG/Tutorial)

* W3C: [SVG Specification](https://www.w3.org/TR/SVG2/)

#### Tools
* **AI**
  * [svgmaker.io](https://svgmaker.io/), $20 for a bunch of credits, has APIs and MCP Server.
* **Editors**:
  * [Boxy SVG](https://boxy-svg.com/) - SVG editor. Paid subscription to save image.
  * [Inkscape](https://inkscape.org/) - Free and open source vector graphics editor. Was unable to install.
  * [Figma](/webdev/figma) - Exports to SVG format
  * [Excalidraw](/webdev/excalidraw) - Exports to SVG format
* **Optimization**:
  * [SVGO](https://github.com/svg/svgo) - Node.js-based SVG optimizer (command-line)
  * [SVGOMG](https://jakearchibald.github.io/svgomg/) - Web-based GUI for SVGO optimization
  * [Vecta.io Nano](https://vecta.io/nano) - SVG compressor with 80% average size reduction
